### How to run the server

```bash
# Create python virtualenv
$ virtualenv venv
# Activate virtualenv
$ source venv/bin/activate
# Install requirements of python package
$ pip install -r requirements.txt
# Run the server
$ python app.py
# URL: localhost
```

### How to run the server unit tests

```bash
$ python app_test.py
```
